5 aplicaciones web más famosas en la actualidad

Instagram

lanzada el 6 de octubre de 2010.
Desarrollador Meta

cuenta con una función llamada Historias (en inglés, Stories) donde todas las personas pueden publicar fotografías y vídeos temporales a su perfil


las tecnologias usadas fueron:

  • Django en Amazon High-CPU Extra Large
  • servidor WSGI a Gunicorn,
  • Fabric
  • PostgreSQL
  • repmgr

facebook

lanzada el 6 de octubre de 2010.
desarrollado por Mark Zuckerberg

Los usuarios pueden agregar a otros usuarios como «amigos», intercambiar mensajes, publicar actualizaciones de estado, compartir fotos, vídeos y enlaces, usar varias aplicaciones de software (apps) y recibir notificaciones de la actividad de otros usuarios.

las tecnologias usadas fueron:
  • LAMP
  • GNU/Linux
  • Java.
  • Hack
  • Erlang



tiktok

lanzada el 6 de octubre de 2010.
Desarrollador ByteDance

Los usuarios pueden hacer una variedad de videos de formato corto y vertical, desde géneros como danza, comedia y educación, etc., que tienen una duración de 1 segundo, hasta 10 minutos.

las tecnologias usadas fueron:
  • AWS Cloudfront: para CDN y almacenamiento en caché
  • API Gateway: como un servicio completamente ad/*6*/inistrado para alojamiento de APIs
  • AWS Lambda: plataforma de cómputo serverless, para ejecutar el código de la aplicación
  • AWS S3: para alojamiento de imágenes
  • AWS CDK for Infrastructure as Code

whatsapp

lanzada el 6 de octubre de 2010.
desarrollado WhatsApp LLC (Meta Platforms, Inc.)

La aplicación permite enviar y recibir mensajes mediante Internet, además de imágenes, videos, audios, grabaciones de audio (notas de voz), documentos, ubicaciones, contactos, gifs, stickers, así como llamadas y videollamadas con varios participantes a la vez

las tecnologias usadas fueron:
  • protocolo abierto Extensible Messaging and Presence Protocol.
  • hash MD5
  • dirección MAC
  • servidor HTTP
  • Base64

waze

lanzada el 6 de octubre de 2010.
Desarrollador Waze Mobile

a diferencia de los softwares de navegación asistida por GPS tradicionales, este es mantenido por los usuarios y aprende de las rutas recorridas por sus usuarios para proveer información de enrutamiento y actualizaciones de tráfico en tiempo real.

las tecnologias usadas fueron:
  • geolocalizacion
  • big data
  • sistema de trafico
